# 技术环境与依赖清单

> **目的**：记录本项目使用的所有技术、框架、依赖库及其版本，避免更换开发环境时因版本不一致导致问题。  
> **最后更新**：2026-05-31  
> **项目名称**：写作助手 (writing-assistant)  
> **应用类型**：Tauri v2 桌面应用（前端 Vue 3 + 后端 Rust）

## 八、版本锁定文件

以下文件记录了确切的依赖版本（锁定文件），**应纳入版本控制**：

| 文件 | 用途 | 是否提交 |
|------|------|---------|
| `package-lock.json` | 锁定 npm 依赖版本 | ✅ 是 |
| `src-tauri/Cargo.lock` | 锁定 Rust 依赖版本 | ✅ 是 |

> ⚠️ **更换开发环境后的标准操作流程**：
> 1. 安装对应版本的 Node.js、Rust、Python
> 2. 执行 `npm ci`（精确安装 package-lock.json 中的版本，不要用 `npm install`）
> 3. Cargo 会自动根据 `Cargo.lock` 恢复版本
> 4. 如遇 WebView2 相关问题，确保 Windows 已更新到最新版
